## How to install

 - Open your terminal where the file is located and type
 ```bash
  pip install -r requirements.txt
```
 - Wait untill it done

 # How to run

 - Open your terminal where the file is located and type
 ```bash
  Python3 scanner_window.py
```
 - Enjoy!
